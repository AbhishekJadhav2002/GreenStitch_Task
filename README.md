# GreenStitch Task



## Steps to run the project
##### 1. Clone the project
```bash
git clone https://github.com/AbhishekJadhav2002/GreenStitch_Task.git
```
##### 2. Install the dependencies
cd into the project directory and run the following command
```bash
cd GreenStitch_Task
```
```bash
npm ci
```

##### 3. Run the project
```bash
npm run dev
```
